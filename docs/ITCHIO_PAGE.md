## Full description

```
You found the entrance. You always do.

The Goldmaze is a 9-floor dungeon built on forks, traps, and the relics you carry out — or don't.
Each run is a different path through 80+ rooms. No two floors feel the same.

BEAT THE MAZE
───────────
→ Navigate 9 floors of branching rooms
→ Survive 3 boss encounters (floors 3, 6, 9)
→ Collect relics that warp your stats and playstyle
→ Die. Learn. Go deeper.

THE RELICS
──────────
8 relics. Each one changes something about how you move through the maze.
Some make you reckless. Some make you careful. None of them are safe.

SECRETS
───────
There are things in this maze that aren't in the readme.
You'll know them when you find them.

BUILT IN
───────
PICO-8 · 128×128 · Lua · Runs in browser
```
